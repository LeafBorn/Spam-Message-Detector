# 📩 Spam Message Detector

A Machine Learning project that classifies messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP).

---

## 🚀 Overview

Spam messages are unwanted and often harmful. This project builds a system that automatically detects spam messages using text data.

The model processes input text, extracts features, and predicts whether the message is spam or not.

---

## 🧠 Features

- Classifies messages into:
  - ✅ Ham (Not Spam)
  - ❌ Spam
- Uses NLP techniques for preprocessing
- Supports multiple ML algorithms
- High accuracy and precision
- Easy integration into applications
